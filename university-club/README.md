# UNIVERSITY CLUB — дизайн-материалы

Айдентика и SMM-материалы клуба Hair Expert University.
Источник: [Figma — HAIR EXPERT UNIVERSITY, страница клуба](https://www.figma.com/design/lSQ1fPwmbQjt2RCquTFm2U/HAIR-EXPERT-UNIVERSITY?node-id=2259-28) + печатные макеты.

## Структура

```
logo/            Герб клуба (шахматный щит HSEC + «club») в 4 вариантах
elements/        Графические элементы: связка А→Б, рамка аватара, блик, плашки
templates/       Шаблоны постов: недели 1–4, расписания месяцев, программа выезда,
                 доп. задание, вакансии, «Символы Мастерства», поздравления, итоги игры
cards/teams/     Карточки команд клубной игры (Щучка, Ярко, Color Мафия, ПАЗЛ,
                 Леди боссы, Без Потолка, HairCash) — составы, капитаны, результаты А→Б
cards/members/   Именные карточки участниц: результаты «точка А → точка Б»,
                 «Что внедрила», лидеры недели
stories/         Сторис 1080×1920 (стратегические сессии) — png
banners/         Баннеры клуба (готовые)
pattern.pdf      Паттерн клуба
club-cards.pdf   Карты клуба · notebook.pdf Блокнот · card-box.pdf Коробка для карт
```

## Стиль клуба

- Фон — глубокий тёмно-синий (navy), высокий контраст.
- Акцент — золото/бронза с градиентом; редкие световые блики.
- Герб: шахматный щит с готическими литерами H/S/E/T + рукописное «club».
- Заголовки — антиква с каллиграфическими буквицами (стиль Cormorant),
  подписи и данные — строгий гротеск.
- Декоративная рамка-паттерн (золотая геометрия) в сторис и печатных макетах.
- Система «точка А → точка Б» — фирменный визуальный приём для результатов участниц.

## Примечания

- Дубликаты из выгрузки Figma удалены (196 → 138 уникальных файлов).
