# HEU Templates · Hair Expert University

Эталонные шаблоны и ассеты для производства брендированных PDF-материалов HEU.
Все PDF собираются через **WeasyPrint** из HTML+CSS.

## Как пользоваться (с Claude)

В новом чате написать, например:

> Сделай конспект деканской сессии по этому транскрипту.
> Шаблон: https://github.com/ИМЯ/heu-templates/tree/main/dean-sessions

Claude скачает шаблон, шрифты и ассеты из репозитория и соберёт PDF по правилам.

## Структура

| Папка | Что это | Формат |
|---|---|---|
| `dean-sessions/` | Конспекты «Разбор с деканом» | А5, crimson-обложка |
| `strategic/` | Конспекты стратегических сессий | А5, тот же брендбук HEU |
| `club/` | Конспекты HEU Club (суббренд!) | А5, navy/gold |
| `usova/` | «Геометрия формы» Марии Усовой | 154×216 мм, монохром |
| `homework/` | Домашние задания №1–7 | А5, тёмная обложка |
| `fonts/` | Склеенные шрифты (latin+cyrillic) + fonts.css | woff2 |
| `assets/` | Логотипы и извлечённые изображения | png |

## Брендбук HEU (основной)

- Палитра: фон `#EDECEA`, карточки `#FFFFFF`, **crimson `#9B1D34`** (акцент, обложка, футер)
- Факультеты D/S/T/E: `#2D6A4F` / `#1E3A5F` / `#9B1D34` / `#B5621E`
- **Золота НЕТ** (золото — только в суббренде Club)
- Фирменная 4-цветная полоса D/S/T/E: верх обложки + низ футера
- Шрифты: Cormorant Garamond (заголовки, крупные цифры, цитаты курсивом) + Jost 300 (текст)
- Призрачные номера разделов; метрики — Cormorant Garamond crimson
- Сетки данных: рамка 1px, зазор 1px, белые ячейки
- Цитаты: левая линия 3px crimson, курсивная антиква

## Суббренд HEU Club

Navy `#0F2432` + gold `#B78B5A` + deep-gold `#9A6B3C`, кремовый контент `#F4EFE6`.
**Crimson не используется.** Подробности — в `club/INSTRUCTIONS.md`.

## Технические правила (WeasyPrint)

1. Шрифты подключать только из `fonts/` — это склеенные latin+cyrillic файлы.
   Обычные субсеты с `unicode-range` в WeasyPrint ломают кириллицу.
2. У Cormorant Garamond цифры старостильные — для номеров и метрик обязательно
   `font-feature-settings: "lnum" 1`.
3. Проверка результата: `pdftoppm -png -r 60` и визуальный осмотр страниц.
