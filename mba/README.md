# MBA — HEU: дизайн-материалы

Материалы программы MBA Hair Expert University (экспорт из Figma + логотипы).

## Структура

```
logo/             Логотипы MBA (main / center / mini / symbol; light / dark / bw) — svg + png
concept/          Концепция айдентики MBA — 11 слайдов (svg; слайды 08 и 11 — png из-за веса)
banners/          Баннеры и посты (91 шт.: 1000×1000 и 1080×1350)
guides/teaching/  Гайд по преподаванию — 121 страница (png 1200×1698)
program/          Программа курса — карточки-посты 1080×1080
  elements/       Плашки и разделители
  icons/          Иконки
schedule/         Расписание курса MBA — pdf + png @2x (4 версии)
style/            Паттерн, шрифтовые стили (Text Styles), варианты логотипа из Figma
```

- Дубликаты из выгрузки Figma удалены (362 → 333 уникальных файла).

  из-за лимитов GitHub — лежат в iCloud: «MBA Печатные материалы / МВА фигма / концепция».
- Печатные исходники (INDD тетради и др.) — в папке «MBA Печатные материалы» в iCloud.
