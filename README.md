# HEU Templates — брендбук Hair Expert University

Web-ready бренд-активы HEU: логотипы, гербы, факультетские знаки, паттерны, шрифты и финальные макеты.

## Структура

```
heu/                      Основной бренд Hair Expert University
  logo/                   Логотипы HE (4 варианта) — svg / png / pdf
  crest/                  Гербы HE-U (5 вариантов, center и left композиции) — svg / png / pdf
  icon/                   Иконки (3 варианта) — svg / png / pdf
  faculties/              Знаки факультетов D, E, S, T — по 3 версии (color / black / gray)
  patterns/               Фирменные паттерны D, E, S, T и University — svg + png 1080×1920
  brand-elements-part1.pdf  Сводный PDF элементов фирменного стиля (стр. 1–9)
  brand-elements-part2.pdf  Сводный PDF элементов фирменного стиля (стр. 10–12)
mba/
  logo/                   Логотипы MBA — HEU (main / center / mini / symbol; light / dark / bw) — svg + png
university-club/
  banners/                Баннеры клуба (png)
  pattern.pdf             Паттерн клуба
  club-cards.pdf          Карты клуба (финальный макет)
  notebook.pdf            Блокнот клуба (финальный макет)
  card-box.pdf            Коробка для карт (финальный макет)
prague-mastermind/
  certificates.pdf        Сертификаты участников
  guest-certificate.pdf   Сертификат гостя
  awards.pdf              Награды (в кривых)
fonts/
  gilroy/                 Gilroy Regular / Medium / Semibold (ttf)
```

## Факультеты

| Код | Суффикс файла | Версии |
|-----|---------------|--------|
| D   | `d-*`         | color, black, gray |
| E   | `e-*`         | color, black, gray |
| S   | `s-*`         | color, black, gray |
| T   | `t-*`         | color, black, gray |

## Примечания

- Исходники (AI, EPS, INDD) не включены — они остаются в рабочей папке дизайнера в iCloud («HAIR EXPERT UNIVERSITY»).
- Шрифт Gilroy — коммерческая лицензия: не публикуйте репозиторий как открытый без проверки условий лицензии.
- Имена файлов транслитерированы для совместимости; соответствие оригиналам — по структуре исходной папки.
